# 🎬 CineVerse – Movie Recommendation System

A full-stack Movie Recommendation System built with FastAPI and React.

---

## 🚀 Features

- Smart movie recommendations
- Live search functionality
- Responsive full-screen UI
- FastAPI backend
- Machine learning similarity model

---

## 🛠 Tech Stack

### Backend
- FastAPI
- Uvicorn
- Pandas
- Scikit-learn
- Pickle

### Frontend
- React JS
- Axios
- Tailwind CSS
